# -Coffee-Sales-Analysis
This project focuses on analyzing Coffee Shop Sales Data to uncover meaningful business insights related to sales performance, customer behavior, and product trends. The analysis helps in understanding revenue patterns, peak sales hours, best-selling products, and overall business growth opportunities.  
The project is built using Microsoft Power BI with structured data sourced from Excel.
